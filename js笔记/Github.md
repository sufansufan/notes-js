# Github

1.在git bash里,

$ git config --global user.name "Your Name"

$ git config --global user.email "email@example.com"

2.进入github登录自己的账号

3.进入home命令台，

4.cd   把文件拖进去

5.git clone 复制克隆的网址

6.cd 复制的那个文件

7.git add  *   提示白色的文件 

8.git status   提示绿色 的文件

9.git commit -m'版本号'

10.git push 回车开始上传；

fatal：致命的错误



设置上传限制

git config http.postBuffer 524288000
